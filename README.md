# IR-control-amp

## Overview
Infra Red remote control amplifier firmware on atmega328p

## Description
このソフトウエアはatmega328p使用  
赤外線リモコンコントロールヘッドホンアンプ基板Rev: A2.x用  
組込みファームウエアです。

## Requirements
Atmel Studio 7 (Version: 7.0.594 - )
